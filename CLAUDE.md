# GOHSY 에이전트 프로토콜

> 이 문서는 Claude Code가 gohsy 레포지토리에서 작업할 때 따라야 하는 가이드입니다.

---

## 1. 프로젝트 개요

### 목적
소원 실현 플랫폼 - 목표 설정 및 달성 추적 서비스

### 기술 스택
- 순수 정적 사이트 (HTML/CSS/JS)
- GitHub Pages 호스팅

### 핵심 가치
- 목표 달성 지원
- 소원 실현 추적
- 개인 성장 동반

## 5. 작업 시 주의사항

1. 수정 전 반드시 `git pull` 실행
2. 커밋 메시지는 한글로 명확하게
3. **Private 레포** - 외부 노출 주의
4. 사용자 개인정보 처리 주의
