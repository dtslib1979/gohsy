04 — Checklist

새 프로젝트 시작 시

  • 1
    Brand Name 정하기 프로젝트의 공식 명칭
  • 2
    Primary Color 선택 골드, 실버, 블루 등 메인 컬러
  • 3
    Background Color 결정 다크/라이트 모드 방향성
  • 4
    Slogan 작성 한 문장으로 브랜드 정의
  • 5
    Hero 콘텐츠 준비 영상 또는 이미지 에셋
  • 6
    고유 콘셉트 아이디어 차별화되는 독창적 요소
  • 7
    UI 메타포 결정 명함? 대시보드? 카드? 포트폴리오?

"위 7개 항목만 채우면,
나머지는 보일러플레이트가 해결한다."

— 10x Production System
