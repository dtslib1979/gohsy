📖 dtslib 매뉴얼

1. 개요

dtslib는 GitHub Pages 기반 웹사이트 플랫폼입니다. AI 업무 자동화를 위한 도구와 템플릿을 제공합니다.

2. 시작하기

  • GitHub 계정으로 로그인
  • 레포지토리 Fork 또는 Clone
  • GitHub Pages 활성화
  • config.json 수정

3. 파일 구조

  • index.html - 메인 랜딩 페이지
  • staff/ - 스태프 관리 포털
  • card/ - 디지털 명함
  • tools/ - 유틸리티 도구
  • config.json - 설정 파일

4. 레포지토리 구성

dtslib 플랫폼은 6개의 레포지토리로 구성됩니다:

  • branch - 메인 사이트 (현재)
  • posts - 블로그 포스트
  • logs - 작업 로그
  • pwa - PWA 앱
  • apk-lab - 안드로이드 앱
  • media - 미디어 CDN

5. Control Panel 도구

  • Image Manager - 이미지 업로드 및 CDN 관리
  • Audio Manager - 오디오 파일 관리
  • HTML Poster - 블로그 포스트 배포
  • WebPrompt - AI 프롬프트 생성기
  • Troubleshoot - 문제 해결 체크리스트

6. Claude Code 사용

CLAUDE.md 파일을 참고하여 Claude Code와 함께 작업할 수 있습니다.

  • 파일 수정 규칙 준수
  • 커밋 메시지 형식 준수
  • 보안 사항 준수
