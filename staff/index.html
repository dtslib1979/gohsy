## 메인 레포지토리 (branch) 구축 GitHub Pages 기반 랜딩페이지 + Staff Portal 레포지토리를 구축해주세요. ### 구조 ``` {username}-branch/ ├── index.html # 랜딩페이지 ├── config.json # 사이트 설정 ├── sw.js # Service Worker (PWA) ├── robots.txt # SEO ├── sitemap.xml # 사이트맵 ├── .nojekyll # Jekyll 비활성화 ├── CLAUDE.md # Claude Code 가이드 ├── assets/ │ ├── manifest.json # PWA 매니페스트 │ └── icons/ │ └── logo.png # 앱 아이콘 ├── staff/ │ └── index.html # Staff Portal (비밀번호: 1126) └── tools/ ├── html-poster.html # HTML 포스팅 도구 └── webpage-prompt.html # 웹페이지 프롬프트 생성기 ``` ### 디자인 시스템 - 다크 테마: 배경 #0a0f0c, 골드 #f0c850 - 모바일 퍼스트 (max-width: 500px) - PWA 지원 (오프라인 캐싱) ### 필수 기능 1. 랜딩페이지: 명함 카드, GitHub Pages 패널, 링크트리 2. Staff Portal: 비밀번호 게이트, 레포 네트워크 대시보드 3. HTML 포스터: GitHub API로 블로그 레포에 HTML 배포 ### GitHub Pages 설정 - Settings → Pages → Deploy from main branch - .nojekyll 파일로 Jekyll 비활성화 이 구조로 레포지토리를 생성하고 커밋해주세요.
